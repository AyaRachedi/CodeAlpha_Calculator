# CodeAlpha_Calculator

A simple calculator built using HTML, Tailwind CSS, and JavaScript as part of the CodeAlpha Frontend Development Internship.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Percentage
- Change Sign (+/-)
- Clear Display (AC)

## Technologies Used

- HTML
- Tailwind CSS
- JavaScript
